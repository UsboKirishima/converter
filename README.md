## How To Install

#### 1st・Installation (Using git CLI)
```
$ git clone git@github.com:UsboKirishima/converter.git
```

#### 2nd・Run (Using curl)
```
$ make

#or

$ gcc allfile -o bin/main
$ ./bin/main
```

#### 2nd・Update (Using git CLI)
```
$ git pull
```
